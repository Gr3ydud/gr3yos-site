Gr3yOS
======

Minimal, modular Linux-based operating system.
Designed to be tiny, stable, and user-hosted.

Downloads
---------

[1] Desktop Edition (Gr3yOS)
    curl -O https://Gr3ydud.github.io/gr3yos-site/gr3yos-latest.iso STATUS: unreleased and bugtesting

[2] Server Edition (Gr3yscale)
    curl -O https://Gr3ydud.github.io/gr3yos-site/gr3yscale-latest.iso STATUS: unreleased

[3] Optional Testing Tools
    curl -O https://Gr3ydud.github.io/gr3yos-site/gr3y-testkit STATUS: WORKING ON IT

Checksums
----------
Verify downloads with:

curl -O https://USERNAME.github.io/gr3yos-site/releases/SHA256SUMS
sha256sum -c SHA256SUMS

Notes
-----
- Minimal, CLI-first OS
- No automatic updates
- Modular
- Experimental builds may have bugs
- Recommended for users familiar with Linux

CLI Users
---------
Use 'curl' or 'wget' to download files.
No browser required.
